About BirdF2L

BirdF2L is a speedcubing algorithm database similar to Birdflu, but instead of Last Layer (LL) algorithms, we are interested in First Two Layer (F2L) algorithms.

BirdF2L was written by Andrew Robbins, with the speed calculation written by Lars Petrus.
