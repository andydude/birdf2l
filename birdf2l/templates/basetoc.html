BirdF2L (Advanced2)

There are already advanced F2L resources available elsewhere (for example: SpeedSolving, CubeFreak (has image problems), Opticubes (has existence problems), JPerm's YouTube Channel, etc.). This is a little more advanced because we cover all 167 cases including OLL (Vj). We use Speffz notation here. Corners are written in uppercase. Edges are written in lowercase. The target slot is always Vj (Front Right), and the Speffz notation is used to specify where each piece comes from. This is the opposite of traditional LL (Last Layer) algorithm notations, which specify where the pieces are going.
